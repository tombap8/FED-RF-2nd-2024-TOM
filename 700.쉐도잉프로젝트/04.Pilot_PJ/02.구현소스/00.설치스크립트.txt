[ Pilot PJ 설치스크립트 ]
1. 리액트 앱 설치하기
- 설치위치: 04.Pilot_PJ\02.구현소스
- 설치이름: pilot-app
- 스크립트: npx create-react-app pilot-app

2. 팩키지 모듈 설치하기
- 설치위치: 04.Pilot_PJ\02.구현소스\pilot-app
- 스크립트: 
    (1) npm i jquery
    (2) npm i jquery-ui-dist
    (3) npm i jquery-ui-touch-punch
    (4) npm i swiper
    (6) npm i scss

3. 설치확인
- 파일명: package.json
- 항목명: "dependencies"

4. 개발서버구동
- 서버구동위치: 04.Pilot_PJ\02.구현소스\pilot-app
- 스크립트:
    (1) 시작: npm start
    (2) 멈춤: ctrl+c 후 질문에 답: y
